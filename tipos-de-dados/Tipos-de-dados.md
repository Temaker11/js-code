# Tipos de dados
Conforme o ECAMAScript standard temos 9 tipos de dados:

* Data types
  * Primitive / Primitive value
  * Structural
  * Strutual Primitive

## Primitivos

* String 
* Number
* Boolean
* undefined
* Symbol
* BigInt

## Estruturais

* Object
  * Array
  * Map 
  * Set
  * Date
  * ... 
* Function

## Primitivo Estrutural / structural root Primitive

* null
